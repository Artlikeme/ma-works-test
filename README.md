
# Test Task for ma.works
Parsing application




## Run Locally

Clone the project

```bash
  git clone "link"
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  python -m venv venv
  source venv/bin/activate
  pip install -r requirements.txt
```

Start the server

```bash
  python main.py
```
